# 2023-Nigeria-Election-Result-Analysis
# overview
This project analyzes the 2023 Nigeria Election Result dataset to explore voting patterns, party performance, and regional trend across the country. the Goal of this analysis is to demontrate data cleaning, visualization and insight extraction using real wprld electoral data. the project focuses on presentation clear,data-driven observation while maintaining neutrality and accuracy.

# Objectives
. Analyze Total Votes by Candidates and party
. Compare voting across states
. Identify regions with the highest and lowest turnouts
. visualize election outcomes using charts and dashboards
# dataset
The dataset was gotten from Vanguard news publication 
# format: CSV 
# Key Columns
. state
. state
. Total Votes
The Data was cleaned and moified
